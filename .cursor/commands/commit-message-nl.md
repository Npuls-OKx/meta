- Samenvatting in 50 tekens of minder (gebiedende wijs).
- Daarna een lege regel.
- Daarna (indien nodig) een toelichting, afgebroken op ongeveer 72 tekens per regel.
- Leg uit welk probleem dit oplost en vooral waarom deze wijziging nodig is.
- Volgende alinea's gescheiden door lege regels.
- Bullet points zijn ok (gebruik '- ' of '* ').

Sluit af met:

- Fixes: #...
- See also: #...

Output alleen de commit message (geen extra uitleg).
